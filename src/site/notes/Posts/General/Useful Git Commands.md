**_Setup:_**

```bash
git config --global user.name username
git config --global user.email email
```

**_Init Repo:_**

```bash
git init
git add .
git remote add origin remote-url <link>
git commit -m "any_message"
git push --set-upstream origin master
```

**_Commit & Push:_**

```bash
git commit -m "any_message"
git push
```
